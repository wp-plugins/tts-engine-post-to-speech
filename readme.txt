Add free, high quality text to speech to your posts (English).

== Description ==

This plugin provides reliable and high quality text to speech functionality(English) to your posts and is completely free. With a single click of a button located underneath each post, the post title and content will be read to the user in a clear, high quality voice. The functionality is also non-instrusive to the blog page with no redirections to other pages.
The JavaScript media player, 'JPlayer' is used to playback the audio in the mp3 format.
The audio is genereated using the TTSEngine.com API service.
This plugin is compatible across all popular browsers (I.E, Firefox, Chrome, Opera, Safari) and
mobile devices.
In the settings page for the plugin there is an option to include a link to 'TTSEngine.com' beneath the listen button. Please enable this option if you are using the plugin and are happy with the service!
The link is also tasteful and non-instrusive.

== Installation ==

Install using WordPress:

1. Log in and go to 'plugins' -> 'Add New'.
3. Search for 'TTSEngine' and hit the 'Install now' link in the results, Wordpress will install it.
4. Activate the plugin.
5. Optional: Display a link to 'TTSEngine.com' beneath the listen button from the plugin settings. 

Install manually:

1. Download the zip file and unzip it. 
2. Open the unzipped folder and upload the entire contents to your `/wp-content/plugins` directory on the server.
3. Activate the plugin through the WordPress 'Plugins' menu.
4. Optional: Display a link to 'TTSEngine.com' beneath the listen button from the plugin settings.
